Free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency. Need to install to PC.

    type git then press enter
    clone github account to PC
git clone https://github.com/username/repositoryname
    to add file into your repository use the following command:
git add . git commit -m "message" git push

CSS
Cascading Style Sheets is used to style and layout the visual presentation of HTML documents. Allows you to control the colors, fonts, spacing, as well as positioning of elements on a web page. Apply either within HTML doc. or externally.

example:

body { font-family: 'Arial'; padding: 15px; } In this code, the sont as well as the padding of the code in HTML doc. is changed through CSS.
Hyperlinks

is a reference to data that the user can directly follow by clicking or tapping. It typically refers to a web page but can also be a reference to an image, a file, or another resource

  • Web Page link a href="link or file path"> /a'
  • img link a href="image link or file path"> /a'

Fundamental structure of the web, enabling navigation between web pages and resources.

Ngrok

A cross-platform command-line tool that enables developers to expose local servers to the internet. It creates secure tunnels from a public endpoint to a locally running web service, making it accessible from anywhere.

  • Make account in ngrok.com
  • After creating , click main menu button
  • Under Cloud Edge , click Domains then create domain.
  • Return to main menu, under getting started, click setup & installation.
  • Under agents, click Linux
  • Copy the "curl -s ...."
  • opem cmd and type wsl
  • paste the link then the ngrok config under static domain
